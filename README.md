### Instructions to run the project (with [Laravel Sail](https://laravel.com/docs/10.x/sail)):
```bash
# Clone the repository and run the following commands
composer install
cp .env.example .env
php artisan key:generate
# Start the docker and run
./vendor/bin/sail up
```
